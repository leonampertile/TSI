Regra de Cramer

A regra de Cramer é uma das maneiras de resolver um sistema linear, mas só poderá ser utilizada na resolução de sistemas que o número de equações e o número de incógnitas forem iguais. Portanto, ao resolvermos um sistema linear de n equações e n incógnitas para a sua resolução devemos calcular o determinante (D) da equação incompleta do sistema e depois substituirmos os termos independentes em cada coluna e calcular os seus respectivos determinantes e assim aplicar a regra de Cramer que diz:

Os valores das incógnitas são calculados da seguinte forma:

X1 = D1/D, X2 = D2/D, X3 = D3/D ... Xn = Dn/D

Veja no exemplo abaixo de como aplicar essa regra de Cramer:

Suponha que você faça orçamentos de 3 tipos de carros para compor sua revendedora. Você faz orçamentos com quantidades diferentes. A concessionária iFcar não dá os valores unitários, pois isso promove a concorrência. Assim, você faz três orçamentos:

Orçamento 1: 3 iF Machine S, 8 iF Machine L, 5 iF Machine SL, equivale a R$ 1.210.000,00

Orçamento 2: 2 iF Machine S, 10 iF Machine L, 3 iF Machine SL, equivale a R$ 1.100.000,00

Orçamento 3: 5 iF Machine S, 3 iF Machine L, 12 iF Machine SL, equivale a R$ 1.660.000,00

Quanto custa cada carro?

Notemos que é possível montar um sistema linear onde cada orçamento é uma equação linear e x, y, e z representam os respectivos modelos S, L e SL do iF Machine. Assim:

Pela regra de Cramer:

Primeiro calculamos as determinantes

D = (3 x 10 x 12) + (8 x 3 x 5) + (5 x 2 x 3) - (5 x 10 x 5) - (8 x 2 x 12) - (3 x 3 x 3) = 41

Dx = (1210000 x 10 x 12) + (8 x 3 x 1660000) + (5 x 1100000 x 3) - (5 x 10 x 1660000) - (8 x 1100000 x 12) - ( 3 x 3 x 1210000) = 2.050.000

Dy = (3 x 1100000 x 12) + (1210000 x 3 x 5) + (5 x 2 x 1660000) - (5 x 1100000 x 5) - (1210000 x 2 x 12) - (3 x 3 x 1660000) = 2.870.000

Dz = (3 x 10 x 1660000) + (8 x 1100000 x 5) + (2 x 3 x 1210000) - (1210000 x 10 x 5) - (8 x 2 x 1660000) - (3 x 1100000 x 3) = 4.100.00

Então temos que:

D = 41
Dx = 2.050.000
Dy = 2.870.000
Dz = 4.100.000

Por ultimo, dividimos as determinandes Dx, Dy, Dz pela determinante D

Dx/D = 2.050.000 / 41 = 50.000
Dy/D = 2.870.000 / 41 = 70.000
Dz/D = 4.100.000 / 41 = 100.000

Como resultado temos: x = 50.000; y = 70.000; z = 100.000

R: O iF Machine S custa R$ 50.000,00, o iF Machine L custa R$ 70.000 e o iF Machine SL custa R$ 100.000,00.
